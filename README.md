## About

**NovaMusic** is a custom HTML wallpaper for **Wallpaper Engine** & **Lively Wallpaper** that displays the current song title, cover art, artist name, and synced animated lyrics in a clean, modern, and beautiful style.

---

## Installation

### Wallpaper Engine:
1. Download the files in [releases](https://github.com/TheYali1/NovaMusic/releases).
2. Extract the zip file in `C:\Program Files (x86)\Steam\steamapps\common\wallpaper_engine\projects\myprojects`
3. Open Wallpaper Engine & Select NovaMusic
4. Done!

### Lively Wallpaper:
1. Download the files in [releases](https://github.com/TheYali1/NovaMusic/releases).
2. Grab the zip and put it in Lively Wallpaper
3. Done!
